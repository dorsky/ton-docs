## Primary Bug Bounty and Crowdsourced Audits Platform

[hackenproof.com](HackenProof), a leading web3 bug bounty and crowdsourced audits platform, hosts official [https://hackenproof.com/company/ton/programs] (TON's bug bounty programs).

:::info
To bolster the security landscape of the TON Blockchain and Ecosystem, TON Foundation enhances the accessibility of bug bounty programs with [https://eco.ton.org/en/opportunities/hackenproof-bug-bounty-program] (tailor-made launching conditions) which work for each TON-based project (including any stack like smart contracts, mini apps, dApp, SDK, API, web, mobile).
